# Routing Rules

## Route to Child Skills

### Use `wtfgames-casino-growth`

- Casino offer copy
- Waitlist or bonus conversion messaging
- Deposit-funnel experiments
- Crypto-casino positioning

### Use `wtfleagues-content-growth`

- Hook ideation
- Episode format design
- Channel-specific content calendars
- Viral clip packaging and social publishing systems

## Keep Request in Orchestrator

- Any prompt combining both sides in one plan
- Any request requiring shared KPI tradeoffs
- Any cross-funnel campaign design (content -> casino conversion)
- Any social audit spanning multiple channels/accounts

## Dry-Run Trigger Cases

- "Give me 15 hooks for hamster livestream clips" -> `wtfleagues-content-growth`
- "Rewrite this casino bonus copy for higher signup intent" -> `wtfgames-casino-growth`
- "Build one campaign linking live clips to casino signups" -> `wtf-growth-orchestrator`
