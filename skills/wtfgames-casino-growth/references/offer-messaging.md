# Offer Messaging System

## Core Angles

- Scarcity angle: early-access framing.
- Status angle: "be early" community positioning.
- Reward angle: cashback/bonus value framing.
- Spectacle angle: tie to the wild live-show identity.

## Copy Block Template

Use this order:

1. Hook
2. Core promise
3. Offer mechanic
4. Action CTA

## CTA Library

- "Sign up for early access"
- "Claim launch access"
- "Join Discord + get first-drop alerts"
- "Get on the list before launch"

## Objection Blocks

- Trust objection: emphasize transparent terms and staged rollout.
- Complexity objection: emphasize one-step signup flow.
- Timing objection: emphasize limited pre-launch window.

## Quality Checks

- Keep sentence length short for social contexts.
- Keep one primary action only.
- Do not bury the offer mechanic in brand fluff.
